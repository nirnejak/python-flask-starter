# HackOn Hackathon

HackOn Hackathon App Backend

## Installation and Setup

**1. Install Required Packages**

```bash
pip3 install pipenv
```

**2. Install Dependencies**

```bash
pipenv install
```

**3. Setup Pre-commit**

```bash
pre-commit install
```

**4. Start Virtual Environment**

```bash
pipenv shell
```

**5. Start Server**

```bash
python3 app.py
```
